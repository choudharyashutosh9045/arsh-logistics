⚠️ Firebase Setup Baaki Hai — Sirf 5 Minute!

Real-time sync ke liye ek baar Firebase setup karo. Neeche diye steps follow karo, phir is file mein FIREBASE_CONFIG fill karo:

1
firebase.google.com kholo → "Go to Console" → "Add project" → Naam daalo (e.g. arsh-logistics) → Continue karo
2
Project open karo → Left menu mein "Realtime Database" click karo → "Create database" → "Start in test mode" → Enable
3
Project Settings (gear icon) → "Your apps" → Web icon (</>) click → App register karo → Config copy karo
4
Is HTML file ko Notepad/VS Code mein kholo → Upar FIREBASE_CONFIG wala section dhundho → Apni values daalo → Save karo
5
Updated file ko Netlify Drop pe upload karo → Link sab ke saath share karo → Ab sab ka data real-time sync hoga! 🎉
